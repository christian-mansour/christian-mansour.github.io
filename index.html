SwimSpot Austin

SwimSpot Austin allows users to view information and reviews on the most popular swimming holes in Austin, TX. Users may browse reviews written by other users; however, users desiring to create their own reviews must sign-up and login to post, edit or delete a review. Users are only able to perform these actions on their own reviews.

Nativ

Nativ is a travel web application with images that users can select. Each image is associated with concepts. The user is able to view a results page with the top five hotels recommended for them based on the images and their associated concepts. A link to the web application is not provided because Nativ was built during a hackathon and the API key was only for temporary use.

Task Lister

Task Lister allows users to create a list of tasks and view a route that connects each task. The user has the ability to delete a task. This would cause the route to be re-routed, excluding the deleted task.
